Privacy Policy

Nexus is a personal dashboard that aggregates work items and pull requests from connected accounts. This policy describes what data is collected and how it is handled.

Data collection

Nexus stores encrypted Personal Access Tokens (PATs) for connected accounts in your browser's local storage. These tokens are decrypted by the server only when needed for API requests, after which they are immediately discarded. No data is transmitted to or stored on any server beyond the provider APIs you explicitly connect.

Data sharing

No data is shared with third parties. All provider API calls are made directly from the application server on your behalf using the tokens you supply.

Data retention

Tokens are stored only in your browser. Tokens are decrypted by the server and exist on the server only for the life of each API request. Clearing your browser's local storage removes all stored credentials.

Contact

For questions about this policy, open an issue on GitHub.
